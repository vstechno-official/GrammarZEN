API Documentation

Free grammar checking API. Correct text, analyze readability, detect sentiment, and get writing suggestions with a single request.

GET endpoint 3 req / 25 min v1.0.0
Rate Limits

Usage Limits

3 Requests
25 Minutes Window
10K Max Characters
Endpoint
GET /api?req={text}

Check grammar, spelling, and style of the provided text.

ParameterTypeStatusDescription
req string required The text to check. Max 10,000 characters.
Example Request
// Simple GET request
GET /api?req=She dont knows nothing

// cURL
curl "https://your-domain/api?req=She%20dont%20knows%20nothing"

// JavaScript
const res = await fetch(`/api?req=${encodeURIComponent(text)}`);
const data = await res.json();

Response Fields
original_textThe input text as provided
corrected_textText with corrections applied
scoreGrammar score from 0 to 100
issuesArray of detected issues with offsets, messages, replacements
suggestionsWriting improvement tips
readability_scoreFlesch Reading Ease score (0-100)
sentimentpositive, negative, or neutral
rate_limitCurrent rate limit status
Status Codes
200Success - text analyzed
400Missing req parameter or text too long
429Rate limit exceeded (3 req / 25 min)
503Engine still loading
500Internal server error
Rate Limit Headers
X-RateLimit-Limit: 3
X-RateLimit-Remaining: 2
X-RateLimit-Reset: 1500 // seconds until reset (on 429)
